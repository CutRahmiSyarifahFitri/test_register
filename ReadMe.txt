Installation:
1. Before installing taiko and gauge, verify that node.js is installed on your system. If not, please install it first.
2. Then, to install taiko globally use npm install -g taiko
You can verify that taiko is installed by using a command taiko --version
Install gauge using npm install -g @getgauge/cli

Setup a Project:
Step 1: Open VS Code and install gauge extension
Step 2: Open folder test_register
Step 3: run spec in implement.spec
